# Flavor Fusion AI

Flavor Fusion AI is an intelligent culinary and mixology assistant that generates creative recipes and cocktails using local datasets and AI reasoning. Built with Streamlit and designed for serverless deployment using stlite on Vercel, the app runs entirely in the browser while providing advanced filtering, nutrition analysis, and multilingual recipe generation.

## Overview

The system combines traditional data filtering with AI-powered generation. It searches structured recipe datasets, applies user constraints like equipment, cooking time, and skill level, and produces tailored recipes. When enabled, an experimental “Absurd Mode” invents entirely new fusion dishes using unlikely ingredient combinations.

The application also performs offline nutritional analysis using a local food database without relying on external APIs.

## Features

- Hybrid RAG recipe generation using local CSV datasets
- Multi-constraint filtering based on ingredients, equipment, time, and skill level
- AI-generated fusion dishes using an experimental Absurd Mode
- Offline nutritional calculations using fuzzy ingredient matching
- Multilingual recipe output (English, Hindi, Italian, Mandarin)
- Interactive Streamlit interface with dark/light mode toggle
- Serverless deployment using stlite running directly in the browser
- Fast static hosting on Vercel

## Architecture

The system is composed of three main layers.

### 1. FlavorEngine (Core Logic)

The FlavorEngine class handles the backend filtering and recipe discovery logic.

Responsibilities include:
- Parsing recipe datasets
- Filtering recipes using hard constraints
- Matching user ingredients with available recipes
- Selecting suitable results from Indian recipes, global recipes, or cocktail datasets

When Absurd Mode is enabled, the system bypasses the dataset search and instructs the language model to generate a novel fusion dish using statistically unusual ingredient combinations.

### 2. Offline Nutrition Engine

The nutrition system estimates macro nutrients using a local nutrition database.

Steps performed by the algorithm:

1. **Ingredient Matching**  
   The ingredient name generated by the AI is matched to a known food item in the nutrition dataset using fuzzy string matching (Levenshtein distance).

2. **Unit Conversion**  
   Volumetric measures such as cups or teaspoons are converted to grams using a density lookup table.

3. **Macro Calculation**

The total nutritional value is calculated using:

T = Σ (Wi / 100 × Ni)

Where:
- Wi = weight of ingredient in grams  
- Ni = nutritional values per 100g from the dataset  
- T = total macro value for the recipe

The system computes:
- Calories
- Protein
- Carbohydrates
- Fat

All calculations run locally without any external API.

### 3. Interactive UI (Streamlit)

The user interface acts as a control dashboard.

#### Sidebar Controls
- Cooking skill level
- Available kitchen equipment
- Language selection
- Absurd Mode toggle
- Theme switch (Dark / Light)

#### Main Dashboard
- Ingredient input
- Generated recipe cards
- Preparation steps
- Nutritional breakdown

The dark mode feature dynamically changes theme settings using Streamlit configuration injected through session state.

### 4. Multilingual Output

All reasoning and filtering occur internally in English. The final recipe instructions and UI labels are translated to the selected language:

- English
- Hindi
- Italian
- Mandarin

This ensures consistent reasoning while allowing localized output.

## Dataset Requirements

Create a `data` folder and place the following CSV files inside it.

### Recipes

1. **Indian Recipes Dataset**
   - ~5000 recipes
   - Ingredients
   - Instructions
   - Cooking time
   - Equipment

2. **Global Recipes Dataset**
   - Large international dataset (~2M entries)
   - Diverse cuisine coverage

### Beverages

3. **Cocktail Dataset**
   - Drink recipes
   - Volumetric ingredient measurements
   - Preparation instructions

### Nutrition Database

4. **Food Nutrition Dataset**
   - Calories per 100g
   - Protein per 100g
   - Carbohydrates per 100g
   - Fat per 100g
